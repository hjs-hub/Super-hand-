# GestureMouse 手势隔空鼠标
基于 MediaPipe + OpenCV 实现的摄像头手势控制鼠标工具，仅识别右手，支持光标移动、捏合长按拖拽、画面防抖、多摄像头切换，适配16:10显示器。

## 项目简介
调用电脑摄像头识别右手拇指+食指手势，实现无接触操控电脑：
- 双指分开：隔空移动鼠标光标
- 双指捏合：按住鼠标左键拖拽（拖动窗口、文件、绘图等）
内置画面比例自适应裁切、双档位平滑防抖、独立鼠标控制线程，降低误触，支持关闭预览后台持续运行手势识别。

## 环境要求
Python 3.8 ~ 3.11（推荐3.10，MediaPipe兼容性最优）

### 一键安装依赖

```bash
pip install opencv-python mediapipe pyautogui pynput
```
#### 这个依赖有点大，三思而后行。

### 快速运行
安装完依赖后执行

```bash
python f.py
```
程序自动扫描并打开第一个可用摄像头，弹出预览窗口
### 手势操作
1移动光标：伸出右手拇指、食指，两指分开，两指中心点映射屏幕鼠标位置
2长按拖拽：右手拇指食指捏紧，持续捏合触发左键长按；松开手指自动释放左键
### 预览窗口快捷键
按键  |   功能
 Q	 |  完整退出程序，释放摄像头、松开鼠标
 E	 |  开启 / 关闭摄像头预览画面（后台识别不中断）
 1	 |  切换摄像头设备 0
 2	 |  切换摄像头设备 1
 3	 |  切换摄像头设备 2
### 自定义配置（代码顶部全局参数区）
修改 f.py 顶部常量调整操作手感与画面：

`SCREEN_W / SCREEN_H`：显示器分辨率（16:10）

`CAM_RAW_W / CAM_RAW_H`：摄像头采集分辨率 1280×720

`PADDING`：画面四周留白像素，减少边缘误识别

`CLICK_THRESHOLD`：捏合触发距离阈值，数值越小越难触发拖拽

`PINCH_LOCK_FRAME`：连续捏合帧数阈值，防手抖误触

`SMOOTH_MOVE`：普通移动平滑系数，越大越顺滑、延迟越高

`SMOOTH_PINCH`：拖拽模式强防抖系数
